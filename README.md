# 供应链管理系统（SCM）

## 项目简介

供应链管理系统（Supply Chain Management System，简称 SCM）是一套基于 Spring Boot + Vue 3 的企业级信息化管理系统。系统实现了从原材料采购、生产计划、库存管理到销售出库的全流程数字化管理，帮助企业优化供应链流程、提高运营效率、降低运营成本。

## 功能模块

### 核心功能

- **📊 仪表盘** - 数据统计与可视化展示，实时掌握企业运营状况
- **👥 用户管理** - 用户账号的增删改查、角色分配与权限管理
- **📦 物料管理** - 物料信息管理、物料分类与库存预警
- **🏭 产品管理** - 产品信息管理、产品分类与价格体系
- **🚚 供应商管理** - 供应商信息管理、供应商评估与选择
- **📋 库存管理** - 库存查询、库存盘点、库存预警与调拨
- **📅 生产计划** - 生产计划制定、执行跟踪与进度管理
- **💰 销售管理** - 销售订单管理、销售数据统计与分析
- **🛒 订单管理** - 订单处理、订单跟踪与售后服务
- **⚙️ 系统管理** - 菜单管理、角色权限管理、字典管理

### 模块说明

| 模块 | 功能描述 |
|------|----------|
| 仪表盘 | 提供关键业务指标的实时展示，包括销售趋势、库存状态、生产进度等 |
| 用户管理 | 实现用户生命周期管理，支持角色权限的灵活配置 |
| 物料管理 | 管理企业生产所需的各种原材料和辅助材料 |
| 产品管理 | 管理企业生产的各类产品，支持产品分类和价格策略 |
| 供应商管理 | 管理和评估供应商，优化采购渠道 |
| 库存管理 | 实时监控库存状态，支持多仓库管理 |
| 生产计划 | 制定和跟踪生产计划，确保按时交付 |
| 销售管理 | 完整的销售流程管理，支持多种销售模式 |
| 订单管理 | 订单全流程跟踪，从下单到交付 |
| 系统管理 | 系统基础参数配置，保证系统稳定运行 |

## 快速开始

### 环境要求

- JDK 17+
- Node.js 18+
- MySQL 8.0+
- Maven 3.6+

### 后端配置

1. **创建数据库**
   ```sql
   CREATE DATABASE scm DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
   ```

2. **修改数据库配置**
   编辑 `backend/src/main/resources/application.yml`：
   ```yaml
   spring:
     datasource:
       url: jdbc:mysql://localhost:3306/scm
       username: root
       password: your_password
   ```

3. **启动后端服务**
   ```bash
   cd backend
   mvn spring-boot:run
   ```

### 前端配置

1. **安装依赖**
   ```bash
   cd frontend
   npm install
   ```

2. **启动开发服务器**
   ```bash
   npm run dev
   ```

3. **访问系统**
   打开浏览器访问：http://localhost:3000/scm/

## 安全特性

- **JWT 认证** - 无状态身份认证机制
- **密码加密** - BCrypt 密码加密存储
- **权限控制** - 基于角色的访问控制（RBAC）
- **请求拦截** - 全局请求拦截与响应处理
- **SQL 注入防护** - MyBatis-Plus 参数绑定
- **XSS 防护** - 输入内容安全过滤
